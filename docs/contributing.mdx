## Ways to contribute

We welcome contributions from the community! Whether you're reporting a bug, suggesting a new feature, or helping with development, your input helps make Jira MCP better for everyone.

## Submitting suggestions

Have an idea for a new feature or improvement? We'd love to hear it!

<Steps>
<Step title="Check existing suggestions">
  Search our [GitHub Issues](https://github.com/xcollantes/jira-mcp/issues) to see if your idea has already been suggested. If it has, you can add your thoughts to the existing discussion.
</Step>

<Step title="Create a new issue">
  If your suggestion is new, [create a new GitHub
  Issue](https://github.com/xcollantes/jira-mcp/issues/new) with the
  "Enhancement" label.
</Step>

<Step title="Provide details">
  Include as much detail as possible:
  - **Use case**: What problem would this solve?
  - **Proposed solution**: How should it work?
  - **Alternatives**: Have you considered other approaches?
  - **Additional context**: Screenshots, examples, or related issues
</Step>
</Steps>

<Tip>
  **Pro tip**: Feature requests with detailed use cases and implementation ideas
  are more likely to be considered.
</Tip>

## Reporting bugs

Found a bug? Help us fix it by providing detailed information.

<Steps>
<Step title="Check for duplicates">
  Search [existing issues](https://github.com/xcollantes/jira-mcp/issues) to see if the bug has already been reported.
</Step>

<Step title="Create a bug report">
  [Create a new GitHub Issue](https://github.com/xcollantes/jira-mcp/issues/new)
  with the "Bug" label.
</Step>

<Step title="Include essential information">
  Please include:
  - **Description**: Clear description of what went wrong
  - **Steps to reproduce**: Step-by-step instructions
  - **Expected behavior**: What should have happened
  - **Actual behavior**: What actually happened
  - **Environment**: Your OS, Python version, Jira instance type, AI tool used
  - **Logs**: Any relevant error messages or logs (remember to redact sensitive information)
  - **Screenshots**: If applicable, screenshots of the issue
</Step>
</Steps>

<Warning>
  **Security issues**: If you discover a security vulnerability, please do not
  create a public issue. Instead, email the maintainers directly or use GitHub's
  security advisory feature.
</Warning>

## Development contributions

Interested in contributing code? Check out our [Development guide](/development) to get started with the codebase.

## Community guidelines

- Be respectful and constructive in your communications
- Use clear, descriptive titles for issues
- Provide sufficient context and details
- Follow up on issues you've created if more information is requested
- Help others by sharing your experiences and solutions

## Get help

Need help or have questions? You can:

- Check the [documentation](/quickstart) for answers
- Search existing [issues](https://github.com/xcollantes/jira-mcp/issues) and [discussions](https://github.com/xcollantes/jira-mcp/discussions)
- Start a [new discussion](https://github.com/xcollantes/jira-mcp/discussions) for questions
